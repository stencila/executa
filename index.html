A test of Executa in the browser. To make this work, in one terminal run an "Executor" listening on ws://localhost:9000:

For example, in this repo, Executa, you could run:

npm run cli:dev -- --ws

Or, you could run Sparkla using this command in that repo:

npm run serve:dev -- --docker

In another terminal, serve this file 

npx serve

Then open http://localhost:5000 and check the console and network tabs while you run the code below.
