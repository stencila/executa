Protected walk

  • walk<NodeType>(root: NodeType, transformer: (node: schema.Node) => Promise<schema.Node>): Promise<NodeType>
    • Defined in src/base/Manager.ts:140

    Do a depth first walk of a node applying a transformation function.

    This function has three characteristics to be aware of

    • recursively traverses all nodes, including the nodes resulting from transformation
    • awaits for the result of transformation of each child, in order
    • depth first

    These characteristics make it suitable for doing an "in order" traversal of a node e.g. executing the nodes in a document in the order that they appear. But in some cases it may be better to use a different function e.g. in cases where you wish to traverse all nodes but the order does not matter.
