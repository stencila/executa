addressToClients

  • addressToClients(addressStr: string, clientTypes: ClientType[]): Promise<Client[]>
    • Defined in src/base/Client.ts:320

    Translate an address string into one or more clients.

    If the address is a "discovery address" e.g. stdio://* then discovery for the transport will be done and zero or more clients returns. For other addresses, a single client will be returned, unless there is an error in the address in which case none will be returned.

    Parameters

    • addressStr: string

      Address string to translate.

    • clientTypes: ClientType[]

      Array of client types available to be translated to.

    Returns Promise<Client[]>
