generateProtocol

  • generateProtocol(id: string, jwt?: string): string
    • Defined in src/ws/util.ts:20

    Generate the Sec-Websocket-Protocol header.

    This is the only header that can be set in the WebSocket constructor and we use it to pass client id and JWT to the server. The generated header follows the recommendations of https://tools.ietf.org/html/rfc6455#section-1.9 with + used as a separator between its parts.

    Parameters

    • id: string

      The id of the client

    • Optional jwt: string

      A JWT

    Returns string
